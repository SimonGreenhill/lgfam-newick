This R script converts the ASJP16 and ASJP16 distances (as proced by the ASJP software included in ASJPSoftware003.zip and available for download from the authors' website at http://email.eva.mpg.de/~wichmann/ASJPHomePage.htm) into a compressed (xz) RData object and optionally also a CSV (TAB-separated) file, to be used later for computing branch lengths for language family trees.
Run asjp62 on the lists16.txt input file to produce the distances matrix to be converted by the R script. If necessary, compile first the asjp62.f file using a FORTRAN compiler.

Notes for ASJP16: ASJP16 (as opposed to ASJP15) seems to produce several erros when processed through the asjp62 software. Therefore I manually replaced some non-ASCII characters in the language descriptors in listss16.txt (resulting in listss16dd.txt). 
Moreover, a handful of languages are exported wrongly by the asjp62 software making the automatic match between their names in the distance matrix file asjp16-dists.txt and the listss16 file impossible. The contents:
  - ReadMe.txt: this readme file;

  - process-asjp15-distances.r and process-asjp16-distances.r: the R script doing the conversion;

  - ASJPSoftware003.zip: the original software, data and manuals (included for easiness; please consult the http://email.eva.mpg.de/~wichmann/ASJPHomePage.htm for info and updates);
  - listss15.txt.tar.xz and listss16-and-listss16dd.tar.xz: the compressed (xz) file containing the ASJP15 (original) and ASJP16 (original and manually edited) language info; must be extracted before running the R conversion script;
  - asjp15-dists.txt.tar.xz and asjp16-dists-dd.txt.tar.xz: the compressed (xz) distances generated by the ASJP software from ASJP15 and ASJP16 (and manually edited); must be extracted before running the R conversion script;

  - conversion.log: the conversion log, detailing what decisions were made for language duplicates or missing languages;
  - asjp15-dists.RData and asjp16-dists.RData: the ASJP15 and ASJP16 distances as a compressed (xz) R object;

  - gpl-2.0.txt: the GPLv2.0 licence (please note that ASJP software is goverend by Creative Commons Attribution 3.0 [CC BY 3.0], http://creativecommons.org/licenses/by/3.0).
